A CLI Linear Algebra Calculator

Our calculator is intended for personal use rather than enterprise use. 
It is easy to use for those who are beginners to linear algebra and can do the fundamental functions involved (with the exception of eigenvalues and eigenvectors). 
The possible functions of our calculator are scalar multiplication, Gaussian elimination, matrix addition, 
matrix subtraction, matrix multiplication, and inverse. This version cannot do determinant calculations, 
eigenvalues or eigenvectors. With the available functions, our calculator can be used to combine and compare data, 
modify matrices in linear transformations, normalize data, adjust the magnitude of vectors, do transformations, 
represent complex systems’ states, and solve linear systems.  

Currently, the calculator is limited to working with only two matrices simultaneously. 
Expanding this to handle multiple matrices could greatly increase its utility. 
Also, the inclusion of eigenvalue and eigenvector calculations would add to its capabilities, catering to a wider range of mathematical problems.
Another consideration is our approach to memory management. While we currently create new matrices to store results, 
an alternative could involve modifying the original matrices to optimize memory usage. This, however, would need to be balanced 
against the flexibility of performing multiple operations on the same input matrix.
